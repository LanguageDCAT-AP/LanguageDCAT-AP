LanguageDCAT-AP

LanguageDCAT-AP is an application profile initiated in the context of the Common European Language Data Space and aiming to cover the exchange of language data, language models and language technologies among catalogues.

Current official version: release 0.9.2

Structure of the repository

/releases: LanguageDCAT-AP releases (0.9.1, 0.9.2, 0.9.5 (draft)).
